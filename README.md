

Proyek ini merupakan keseluruhan pekerjaan saya selama menjabat sebagai Business Performance, yang mencakup pengelolaan data B2B dan B2C. Saya bertanggung jawab dalam melakukan update performance harian, mingguan, dan bulanan dengan melakukan penarikan data dari sistem OSS, kemudian melakukan pembersihan, pengolahan, serta validasi data untuk memastikan akurasi dan konsistensi data. Saya juga mengelola data dalam jumlah besar dengan menjaga kualitas data agar dapat digunakan secara optimal dalam proses pelaporan.

Awalnya, visualisasi data dilakukan menggunakan PowerPoint dalam bentuk laporan statis. Namun, saya berhasil mengembangkan metode tersebut menjadi dashboard interaktif menggunakan Looker Studio yang lebih informatif dan mudah dipahami. Selain itu, saya melakukan analisis dan visualisasi data untuk menghasilkan insight yang dapat ditindaklanjuti, yang kemudian disajikan kepada stakeholder dan manajemen guna mendukung pemantauan kinerja bisnis B2B dan B2C serta pengambilan keputusan yang lebih cepat dan tepat.
