# **About the Project** 

---
This project represents my overall responsibilities while working as a Business Performance professional, covering both B2B and B2C data management. I was responsible for updating daily, weekly, and monthly performance reports by extracting data from the OSS system, followed by data cleaning, processing, and validation to ensure accuracy and consistency. I managed large datasets and maintained high data quality to support reliable reporting.
Initially, data visualization was done using PowerPoint in the form of static reports. However, I successfully improved this process by developing interactive dashboards using Looker Studio, making the data more informative and easier to understand. I also analyzed and visualized data to generate actionable insights, which were presented to stakeholders and management to support monitoring business performance and making faster, data-driven decisions.

# **Indonesian Ver** 

---
Proyek ini merupakan keseluruhan pekerjaan saya selama menjadi Business Performance, yang mencakup pengelolaan data B2B dan B2C. Saya bertanggung jawab dalam melakukan update performance harian, mingguan, dan bulanan dengan melakukan penarikan data dari sistem OSS, kemudian melakukan pembersihan, pengolahan, serta validasi data untuk memastikan akurasi dan konsistensi data. Saya juga mengelola data dalam jumlah besar dengan menjaga kualitas data agar dapat digunakan secara optimal dalam proses pelaporan.

Awalnya, visualisasi data dilakukan menggunakan PowerPoint dalam bentuk laporan statis. Namun, saya berhasil mengembangkan metode tersebut menjadi dashboard interaktif menggunakan Looker Studio yang lebih informatif dan mudah dipahami. Selain itu, saya melakukan analisis dan visualisasi data untuk menghasilkan insight yang dapat ditindaklanjuti, yang kemudian disajikan kepada stakeholder dan manajemen guna mendukung pemantauan kinerja bisnis B2B dan B2C serta pengambilan keputusan yang lebih cepat dan tepat.
